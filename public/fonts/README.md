# Font Files Required

## Download Instructions

### Satoshi Font (Primary UI Font)

1. Visit: https://www.fontshare.com/fonts/satoshi
2. Download the font package
3. Extract and copy these WOFF2 files to this directory:
   - `Satoshi-Regular.woff2` (Weight: 400)
   - `Satoshi-Medium.woff2` (Weight: 500)
   - `Satoshi-Bold.woff2` (Weight: 600)

### Clash Display Font (Marketing Headings Only)

1. Visit: https://www.fontshare.com/fonts/clash-display
2. Download the font package
3. Extract and copy these WOFF2 files to this directory:
   - `ClashDisplay-Semibold.woff2` (Weight: 600)
   - `ClashDisplay-Bold.woff2` (Weight: 700)

## Final File Structure

```
public/fonts/
├── Satoshi-Regular.woff2
├── Satoshi-Medium.woff2
├── Satoshi-Bold.woff2
├── ClashDisplay-Semibold.woff2
└── ClashDisplay-Bold.woff2
```
